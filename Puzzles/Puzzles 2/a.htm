Word & Letter Play
 
A? 		 
If you start with the number one and use only integers (whole numbers), how far do you have to count before you need to use the letter a in spelling a number?


 
You have to count up to one thousand.

Source: Based on a puzzle in More Puzzles for Pleasure and Leisure by Thomas L. Hirsch. New York: Abelard-Schuman, 1974.
