Listen to the Band 		 
Five friends play together in a band. Each one plays a different instrument. The instruments they play are saxophone, trumpet, clarinet, flute, and drums. Their names are Rosa, Kim, Julie, Keith, and Lily. Can you figure out who plays which instrument from the following four clues?
  • Rosa plays the sax.
  • Kim does not play a wind instrument.
  • Julie does not play the flute or the clarinet.
  • Keith's instrument is held sideways when it is played.


 
Rosa plays the sax, Kim plays drums, Julie plays trumpet, Keith plays flute, and Lily plays clarinet.

  Rosa Kim Julie Keith Lily
saxophone X X X X
trumpet X X X X
clarinet X X X X
flute X X X X
drums X X X X

Start filling in the information you've been given. Then continue filling in information you can deduce from what you already know.

  • Check off the saxophone box in Rosa's column.
  • Write an X in each of the other boxes in Rosa's column and each of the other boxes in the saxophone row.
  • Check off the drums box in Kim's column, since all the other instruments are wind instruments.
  • Write an X in each of the other boxes in Kim's column and each of the remaining boxes in the drums row.
  • Write an X in the flute box in Julie's column. Write an X in the clarinet box, as well.
  • Now you know Julie plays the trumpet, so check off the trumpet box for Julie, and write an X in each of the remaining boxes in the trumpet row.
  • Check off the flute for Keith, since the flute is the only instrument that is held sideways.
  • Write an X in the clarinet box in Keith's row.
  • Write an X in the flute box in Lily's row, since the flute is Keith's instrument.
  • That means Lily plays the clarinet
